# PicklePlay

PICKLEPLAY — Your New Pickleball Playground. Get ready to serve, volley, and smash your way to victory at the heart of Cebu's newest and most thrilling pickleball destination.

## Project Structure

This monorepo contains both the frontend and backend applications:

```
PicklePlay/
├── frontend/          # Next.js frontend application
└── backend/           # Backend API (coming soon)
```

## Frontend

The frontend is built with Next.js 16 (App Router), TypeScript, and Tailwind CSS.

**Quick Start:**

```bash
cd frontend
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the application.

For more details, see [frontend/README.md](./frontend/README.md)
